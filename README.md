## Principais projetos

### 1. My Expenses

Aplicação web para controle financeiro pessoal, com foco em praticidade, organização de gastos, ganhos, categorias, autenticação e experiência mobile.

**Destaques do projeto:**

- aplicação com **front-end e back-end**
- interface moderna e responsiva
- controle de gastos e ganhos
- categorias personalizadas
- autenticação de usuários
- deploy web
- base evoluindo para produto real

**Tecnologias:** TypeScript, React/Next.js, Python, API, deploy, autenticação.

🔗 Repositório: [my_expenses](https://github.com/BMatheus1/my_expenses)

---

### 2. Detecção de Sepse com Machine Learning

Projeto de Machine Learning voltado ao apoio à triagem clínica de risco de sepse, com foco em alta sensibilidade, análise de falsos positivos, falsos negativos e interpretabilidade.

**Destaques do projeto:**

- pipeline de Machine Learning aplicado a problema crítico de saúde
- foco em reduzir falsos negativos
- análise de métricas como recall, precision e specificity
- engenharia de features clínicas
- interpretabilidade com SHAP
- documentação com README e Model Card
- API para servir o modelo
- execução com Docker

**Tecnologias:** Python, Pandas, Scikit-learn, XGBoost/CatBoost, SHAP, FastAPI, Docker, Jupyter Notebook.

🔗 Repositório: [projeto_sepse_2.0](https://github.com/BMatheus1/projeto_sepse_2.0)

---

### 3. PDF Reader Inteligente

Aplicação em Streamlit para leitura, busca e recuperação de informações em PDFs, com suporte a busca lexical, semântica e híbrida.

**Destaques do projeto:**

- upload de múltiplos PDFs
- extração de texto por página
- geração de chunks com overlap
- busca lexical
- busca semântica com embeddings
- busca híbrida
- reranking de resultados
- resposta curta com evidências rastreáveis
- cache local para evitar reprocessamento
- estrutura modular preparada para evolução para RAG

**Tecnologias:** Python, Streamlit, PyPDF, NumPy, Sentence Transformers, Pytest.

🔗 Repositório: [pdf_reader](https://github.com/BMatheus1/pdf_reader)
